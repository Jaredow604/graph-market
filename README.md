**Grafo-Cart** es un motor de optimización basado en teoría de grafos diseñado para resolver el problema del consumidor moderno: encontrar la ruta y combinación de compras más barata en diferentes supermercados.

## 📌 El Problema a Resolver

El problema principal es la alta variación de precios de los mismos productos entre distintos supermercados. Comparar manualmente el costo de una lista de compras entera en varias tiendas requiere demasiado tiempo y esfuerzo. Además, existe un reto logístico: decidir si el ahorro en ciertos artículos realmente justifica el costo de gasolina y el tiempo de trasladarse a múltiples sucursales. En resumen, el consumidor se enfrenta a un complejo problema de optimización al intentar equilibrar su presupuesto con su tiempo disponible.
